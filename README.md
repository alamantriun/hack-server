# Hack Project

Proyecto con estructura cliente-servidor.

## Estructura

```
hack-project/
├── servidor/
│   ├── package.json
│   ├── server.js
│   └── logs/
├── cliente/
│   ├── index.html
│   ├── style.css
│   └── script.js
└── README.md
```

## Instalación

1. Navega a la carpeta `servidor`:
   ```bash
   cd servidor
   ```

2. Instala las dependencias:
   ```bash
   npm install
   ```

3. Inicia el servidor:
   ```bash
   npm start
   ```

4. Abre tu navegador y ve a `http://localhost:3000`

## Archivos

- **servidor/server.js**: Servidor Express
- **servidor/package.json**: Dependencias del servidor
- **cliente/index.html**: Página principal
- **cliente/style.css**: Estilos
- **cliente/script.js**: Lógica del cliente
